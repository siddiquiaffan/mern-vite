# MERN Vite Template
----

## Includes
- React (Vite)
- React Router DOM
- Express
- Typescript
- Tailwind css

## Build (Production)
```bash
npm i
node build.js --install --zip
```
#### Build args
```
--install   Installs the packages in both client and server before building it
--start     Starts the application
--zip       Creates a production zip folder 
```


## Run Application (Dev)
```bash
cd client
npm i
npm run dev
```

```bash
cd server
npm i
npm run dev
```
